## A commercial license is required for

- company or organizational production use
- customer-facing deployments
- hosted, managed-service, or SaaS-style use
- resale, white-labeling, sublicensing, or embedding Codexify into a paid
  service
- using Codexify as operational infrastructure for paid work or client delivery

## What this file does and does not do

This file does not set final pricing, support terms, service levels, or
deployment entitlements.

It exists to make the usage boundary clear: visible source and free
individual/non-commercial use do not grant commercial deployment rights.

## Current release posture

Nothing in this file expands Codexify's current public release promise.
Codexify remains a local-first beta with a narrow, proof-driven public install
path. Hosted SaaS availability, enterprise production support, and general
commercial deployment rights require a separate agreement and, where relevant,
separate support commitments.
