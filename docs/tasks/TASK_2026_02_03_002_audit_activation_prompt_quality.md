# Task: Audit Activation Prompt Quality

Date: 2026-02-03

## Purpose
Assess whether activation prompts are specific, reproducible, and aligned with campaign objectives.

## Inputs
- Activation prompts for each task
- Campaign objectives and acceptance criteria

## Steps
1. Check each activation prompt for explicit inputs and outputs.
2. Ensure prompts reference success criteria or acceptance conditions.
3. Verify prompts are aligned with the campaign objective and scope.
4. Flag any ambiguity or missing context.

## Outputs
- A short assessment per task prompt.
- A list of required improvements, if any.

## Acceptance Criteria
- Prompts are unambiguous and actionable.
- Prompts reference required inputs and expected outputs.
- Prompts align with campaign goals.
