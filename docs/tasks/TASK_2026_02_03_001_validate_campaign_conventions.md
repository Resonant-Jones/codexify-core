# Task: Validate Campaign Conventions

Date: 2026-02-03

## Purpose
Confirm campaign and task artifacts follow required naming conventions and contain all mandatory sections.

## Inputs
- Campaign document path
- Task artifact paths
- Naming conventions policy

## Steps
1. Verify campaign doc filename matches required pattern.
2. Verify task artifact filename matches required pattern.
3. Check campaign document includes Objective, Scope, Acceptance Criteria, Risks, and Deliverables.
4. Confirm activation prompt includes context, inputs, outputs, and success criteria.

## Outputs
- A short report listing any violations found.
- Confirmation if all checks pass.

## Acceptance Criteria
- All required sections are present.
- All filenames match the specified patterns.
- Activation prompt is complete and actionable.
